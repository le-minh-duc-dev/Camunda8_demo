# Access Camunda Components

Camunda 8 Run uses **basic authentication** with `demo/demo` for all web interfaces.  
You can log in to all web interfaces using the default credentials:

- **Username:** `demo`
- **Password:** `demo`

## Web Interfaces

| Component | URL |
|-----------|-----|
| Operate   | [http://localhost:8088/operate](http://localhost:8088/operate) |
